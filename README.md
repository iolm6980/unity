# 주제
유니티를 활용한 퍼즐 게임 제작

# 프로젝트 소개  
게임을 시작하면 상단에 이웃하면 안되는 캐릭터들이 나옵니다    
해당 조건에 맞춰 배치되어있는 캐릭터들을 재배치하는 게임입니다  

# 개발 환경
* C#
* UNITY
